# industrial-machine-temperature-monitoring

## 📌 Project Overview

This project simulates the thermal behavior of an industrial machine over 200 operating cycles.

The objective is to demonstrate how data analysis can be applied to detect:

- Temperature threshold violations
- Progressive overheating trends
- Potential failure scenarios

The simulation was developed using Python, focusing on industrial monitoring and predictive maintenance concepts.

---

## ⚙ Technologies Used

- Python
- Pandas
- Matplotlib
- Random (data simulation)

---

## 📊 Features Implemented

✔ Automatic generation of operational data  
✔ Temperature threshold monitoring (85°C limit)  
✔ Detection of critical occurrences  
✔ Trend analysis using temperature variation  
✔ Graphical visualization of machine behavior  

---

## 🧠 Engineering Context

In real industrial environments, overheating may indicate:

- Lack of lubrication
- Mechanical wear
- Overload conditions
- Cooling system inefficiency

This simulation models a progressive degradation scenario to illustrate how data-driven monitoring can anticipate failures before critical breakdown occurs.

---

## 📈 Example Output

The generated graph shows:

- Initial stable operation (~70°C)
- Gradual temperature increase
- Limit exceedance
- Critical operating condition
